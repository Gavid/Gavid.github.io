* Git是基于元数据方式分布式存储文件信息的，它会在每一次Clone的时候将所有信息都取回到本地，即相当于在你的机器上生成一个克隆版的版本库。
* 但有时，我们只想copy部分文件，而不想将整个项目copy下来
* 那么，我们可以将文件的路径copy,到[DownGit](https://minhaskamal.github.io/DownGit/#/home)进行下载
