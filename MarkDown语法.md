## 将markdown内容添加`折叠/展开功能`

```html
<details>
	<summary>title</summary>
	content
 </details>
```

## 图片操作

#### 图片缩放大小控制

```html
<img src="xx.jpg" alt="" style="zoom:50%;" />
```

#### 图片对齐方式设置(设置align为center|left|right)

```html
<div align="right"><img src="xxx.jpg"/></div>
```

## 文字操作

#### 文字居中对齐

```html
<center>xxx</center>
```

#### 文字左对齐：

```html
<p align="left">xxx</p>
```
